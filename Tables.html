Tables in HTML

It is used to allow web developers to arrange data into rows & columns.

Key Elements need to remember :

  1. table - defines the table.
  2. th- Defines table header cell
  3. tr - Defines a table row.
  4. td -Defines a table data cell.
  5. Caption - Adds a caption to the table.
  6. thead - Groups the header content.
  7. tbody - Groups the body content
  8. tfoot - Groups the footer content.

What is Padding?

  1. Cell padding is the space between the cell edges and the cell content.(inside the cell).
  2. By default the padding is set to 0.
  3. To add padding on table cells, use the CSS padding property
  4. To add padding only above the content, use the padding-top property.
  5. Add padding to the other sides with the padding-bottom, padding-left, and padding-right properties.

What is Cell Spacing?

  1. Cell spacing is the space between each cell.
  2. By default the space is set to 2 pixels.
  3. To change the space between table cells, use the CSS border-spacing property on the table element
