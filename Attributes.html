href Attribute:

It is used in "a" tag which defines hyperlink.What Actual href does is ? It specifies the URL of the page the link should go.

Target Attribute:

Basically, If we provide the link in href, by default it will display in current browser window. So If you want to open in another window/browser, we must specify the target attribute for the link.

The target attribute specifies where to open the linked document.It can be done in my ways:

  1. _self - Default. Opens the document in the same window/tab as it was clicked
  2. _blank - Opens the document in a new window or tab
  3. For Ex: With Target Value Visit youtube!
  4. Without target value ; Visit youtube!

src Attribute:

src attribute - It is used to specify the source or location of an external resource. It is commonly used with elements like img,script,iframe,audio,video.There are two ways to specify the URL in src attributes

Key Points about src :

Absolute vs Relative URL's

  1. Absolute URL: Links to an external image that is hosted on another website. Example: src="https://www.w3schools.com/images/img_girl.jpg".
  2. Relative URL : A relative URL is a shortened web address that points to a file (like an image) within the same website. It doesn’t include the full domain name (e.g., https://example.com). Instead, it’s based on the current location of the file you’re working with.
  3. If the URL starts without a slash (e.g., src="img_girl.jpg"); The browser looks for the file in the same folder as the current page.
  4. If the URL starts with a slash (e.g., src="/images/img_girl.jpg"); The browser looks for the file starting from the root folder of the website.
  5. We can also specify the height and width of the images, using height and width aatributes.

alt Attribute:

The required alt attribute for the tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection, or an error in the src attribute, or if the user uses a screen reader.
