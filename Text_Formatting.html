Text Formatting :

It defines how text should be displayed in a web browser.
Today, Let's discuss about most commonly used HTML tags for text formatting...

  1. Headings : Headings are used to define the hierarchy of the text. There are six levels of headings, from h1 (the most important) to h6 (the least important). We have already, see how heading works.
  2. Bold Text : The "b" tag or the "strong" tag can be used to make text bold. The "strong" tag also indicates that the text is of strong importance.
    Let's See Examples : This text is bold.
    This text is also bold and indicates strong importance.
  4. Italic Text : The "i" tag or the "em" tag can be used to italicize text. The "em" tag also indicates that the text is emphasized.
    This text is italic
    This text is emphasized
  5. Underlined Text : The "u" tag is used to underline text. Ex: This text is underlined.
  6. Strikethrough Text : "S" or "del" tag can be used to strike through text. For Ex:
    This text is strikethrough.
    This text is also strikethrough and indicates deletion.
  7. Superscript and Subscript: The "sup" tag is used for superscript text. For Ex: superscript text
    The "sub"tag is used for subscript text. For Example : subscript text.
  8. Line Break: The "br" tag is used to break the line.
  9. Horizontal Rule:The "hr" tag is used to create a horizontal line, often used to separate content.
  10. Preformatted Text:The "pre" tag is used to display text exactly as it is writeen in HTML code,preserving spaces and line breaks. For Ex: 
                This is preformatted text.
                It preserves spaces and line breaks.
  11. Inline Quotations: The "q" tag is used for short inline quotations,For Ex: This is short quote
  12. Highlighted Text: The "mark" tag is used to highlight text. For Ex : highlighted
  13. Small Text: The "small"tag is used to make text smaller.For Ex : smaller
  14. Abbreviations: The "abbr" tag is used to define an abbreviation or acronym. For Ex : The WHO was founded in 1948.
  15. Address: The "address" tag is used to define contact information for the author/owner of document.
    Written by John Doe.
    Visit us at:
    Example.com
    Box 564, Disneyland
    USA
  16. Text Direction : The "bdo" tag is used to override the current text direction. For Ex : This text will be written from right to left.
  17. Inserted and Deleted Text : The "ins" tag is used to underline text to indicate that it has been inserted, while the "del" tag is used to strike through text to indicate that it has been deleted. For Ex : blue ; red
